About

Det:B!는 사이트 주소 혹은 사업자 등록번호를 입력 받아
해당 사이트 사업자 등록 진위여부를 파악하여 알려드립니다.


How



국세청 오픈 API: 국세청에서 제공하는 '사업자등록정보 진위확인 및 상태조회' API 서비스를 통해
온라인 사업자가 현재 등록되어 활동 중인 사업자인지 원격으로 확인 가능합니다.

웹 크롤링: 자바 Jsoup 기술을 통한 웹 크롤링 작업을 통해 사이트 주소를 입력하는 작업만으로
해당 사이트의 등록정보를 확인할 수 있습니다. 입력한 사이트에서 사업자 등록번호를 추출하고,
사업자 등록번호를 국세청 API에 조회합니다. 이는 직접 사업자 등록번호를 알아내 입력해야하는
다른 조회 서비스들과 비교하여, Det:B!가 갖고 있는 강점입니다.
